# Sales Performance Dashboard

An interactive **Power BI** dashboard that analyzes sales performance and turns
raw sales data into clear, decision-ready visuals.

## Overview
The dashboard (`Sales_data.pbix`) presents key sales metrics in one place so
trends and performance can be understood at a glance instead of reading through
raw spreadsheets.

## Tools
- **Power BI** — data modeling, visuals, and interactivity
- Sales dataset (imported into the `.pbix` file)

## What It Shows
> Update this list to match the actual visuals in your dashboard.
- Total sales / revenue over time
- Sales by region, category, or product
- Performance against targets
- Top-performing products or segments

## How to Open
1. Download `Sales_data.pbix` from this repository.
2. Open it in **Power BI Desktop** (free from Microsoft).
3. Interact with the filters and visuals to explore the data.
